MPI 初探

  • MPI_Init(NULL, NULL); // 两个参数保留未来使用

  • MPI_COMM_WORLD // a built-in communicator

每个进程有一个特定的rank

获取进程rank:

int world_rank;
MPI_Comm_Rank(MPI_COMM_WORLD, &world_rank);

获取总的并行进程数：

int world_size;
MPI_Comm_Size(MPI_COMM_WORLD, &world_size)
